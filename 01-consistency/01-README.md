# 一致性系统设计约定

@Author  : ShawnPanda 
@Created : 2020-10-23
@Modified: 2020-11-05

### 一、前言目的
 * 一致优先: 快速拉通开发、运维、测试、产品、运营等相关方；
 * 降低成本: 降低沟通、开发、维护各环节成本；
 * 简单轻量: 总投入时间占比小于10%，效能提升10-50%；
 * 团队认可: 能持续进行，取得大家认可；
